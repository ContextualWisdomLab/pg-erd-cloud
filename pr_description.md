🎨 Palette: ERD 테이블 및 컬럼 편집 기능 구현

💡 What:
- `ReactFlow` 캔버스에 추가된 노드(테이블)를 더블 클릭하여 이름을 수정하거나 관련된 컬럼을 추가, 삭제, 수정할 수 있는 모달 UI 추가
- ERD에서 개별 테이블 노드 및 해당 테이블을 즉시 삭제할 수 있는 기능 추가

🎯 Why:
- 기존 MVP 버전은 스냅샷에서 구조를 읽어오기만 할 뿐 ERD 엔지니어링 도구로서 가장 기본적인 UI 스키마 수정(테이블 및 컬럼의 생성, 수정, 삭제) 기능이 부족했습니다.
- 테이블 및 컬럼의 구조를 능동적으로 편집하여 DDL을 export하거나 LLM 프롬프트 생성용 모델로 사용할 수 있도록 확장성을 보완했습니다.

📸 Before/After:
- Before: 캔버스의 빈 화면 또는 스냅샷에서 읽어온 노드만 보이며 더블 클릭 상호작용 불가능, 테이블 삭제 불가.
- After: 노드를 더블 클릭하여 `테이블 편집` 모달 진입, 테이블 이름/코멘트 변경 및 개별 컬럼 세부 정보(NN, PK 설정 등)를 편집할 수 있으며 테이블 삭제 가능.

♿ Accessibility:
- 컬럼 삭제, 테이블 삭제와 같은 파괴적인 작업에는 `window.confirm`을 사용하여 사용자 확인을 거칩니다.
- 버튼 및 인풋 요소들에 `aria-label` 속성을 추가하여 접근성을 높였습니다.
